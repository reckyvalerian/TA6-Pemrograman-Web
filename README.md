# Tugas Akhir Praktikum Pemrograman Web 6 - Weather Dashboard

Weather Dashboard berfungsi untuk menampilkan informasi cuaca real-time dan ramalan 5 hari berdasarkan lokasi yang dicari pengguna. Aplikasi ini dibangun tanpa framework, menggunakan HTML, CSS, dan JavaScript murni, serta memanfaatkan layanan Open-Meteo API dan Nominatim (OpenStreetMap) untuk data cuaca dan geocoding.
